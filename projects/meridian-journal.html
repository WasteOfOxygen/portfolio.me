Editorial design system for an independent arts and culture publication вҖ” grid architecture, type hierarchy, and print production.

// 01 вҖ” OVERVIEW

Meridian Journal is a quarterly arts and culture publication launched by an independent editorial collective. The brief called for a flexible, grid-based design system capable of holding long-form essays, photography portfolios, and short experimental pieces in equal measure.

The solution centres on a strict 12-column baseline grid and a two-typeface system вҖ” a condensed grotesque for headers and a humanist serif for body text вҖ” giving each issue visual consistency while leaving room for per-section personality shifts.

Print production included CMYK adaptation, bleed management, and cover specification for both newsprint and coated stocks across four different print runs.

// 03 вҖ” PROCESS

STEP 01 EDITORIAL AUDIT Reviewed 12 issues of competing publications. Mapped content types, pacing patterns, and reader flow across formats.
STEP 02 GRID ARCHITECTURE Designed a 12-column baseline grid with three layout modes: essay, portfolio, and hybrid. Tested across A4 and tabloid formats.
STEP 03 TYPE SELECTION Selected and licensed two typefaces. Set optical sizing, tracking rules, and spacing ratios for all hierarchy levels.
STEP 04 PRINT PRODUCTION Prepared press-ready PDFs across four print runs. Coordinated with two print houses in Prague and Berlin.
