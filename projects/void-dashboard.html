// 01 вҖ” OVERVIEW

Void is a data visualisation platform built for infrastructure teams tracking system health, pipeline throughput, and anomaly detection in real time. The design challenge was density: how to present high-information dashboards without triggering cognitive overload.

The solution is a modular component library built on an 8px spatial grid, with three density modes вҖ” compact, comfortable, and spacious вҖ” switchable per user. A strict dark theme reduces eye strain during long monitoring sessions, with colour used exclusively to communicate state.

Accessibility was a primary constraint: all interactive elements meet WCAG AA contrast ratios, with keyboard navigation and screen-reader labels built in from the start.
