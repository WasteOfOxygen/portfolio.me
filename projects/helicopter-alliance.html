// 01 — OVERVIEW

Helicopter Alliance needed a print system that could work across everything — large-format event posters, folded leaflets handed out at the door, trifold info sheets, and smaller promotional flyers. The challenge was building something coherent enough to feel like a brand, but flexible enough to serve wildly different contexts and print runs.

The solution was a grid-first approach: a modular layout system where every format is derived from the same proportional logic. Swap the content, the format snaps into place. Typography is set in a single weight family to keep production costs low and reproduction consistent across uncoated stock, digital print, and offset.

The visual language leans into contrast — heavy black blocks, generous white space, and a restricted use of red as the only accent colour. Ink-friendly by design.

// 03 — PROCESS

STEP 01 FORMAT AUDIT Mapped every touchpoint the client needed — from street posters to hand-distributed flyers — before designing anything. Print specs, quantities, and turnaround times defined the constraints.
STEP 02 GRID SYSTEM Built a modular grid that scales proportionally across all formats. One master grid, every layout derived from it. Cuts production time and keeps brand coherence across formats.
STEP 03 COLOUR & TYPE Two-colour palette — black and red — chosen for offset economy and visual impact. Single typeface family. All weights and sizes locked to a scale so any new format can be templated quickly.
STEP 04 PRESS-READY FILES Delivered bleed-ready PDFs for each format with printer profiles embedded. Separate files for digital and offset runs. Preflight checklists included in the handoff documentation.
